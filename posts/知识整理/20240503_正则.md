# 正则
```
\s ==> space
\w ==> word

/<.*>/ ==> <h1>RUNOOB-菜鸟教程</h1>
/<.*?>/  ==>  <h1>  #非贪婪匹配

exp1(?=exp2)：查找 exp2 前面的 exp1。
(?<=exp2)exp1：查找 exp2 后面的 exp1。
exp1(?!exp2)：查找后面不是 exp2 的 exp1。
(?<!exp2)exp1：查找前面不是 exp2 的 exp1。
```


# 身份证正则
```
^[1-9]\d{5}(18|19|20)\d{2}(0[1-9]|1[0-2])(0[1-9]|[1-2][0-9]|3[0-1])\d{3}(\d|X)$
```

# 手机号正则
```
^1[3-9]\d{9}$
```

# 密码正则
(?=.*[a-z])：至少有一个小写字母。

(?=.*[A-Z])：至少有一个大写字母。

(?=.*\d)：至少有一个数字。

[a-zA-Z\d]+：一个或多个字母或数字。
```
^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)[a-zA-Z\d]+$
```
